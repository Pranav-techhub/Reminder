# ReminderApp

A CLI-based Reminder System using Python with a clean architecture.

---

##  Functional Requirements

### Reminder Object
- ID (Auto-incremented)
- Text (Reminder content)
- is_done (True/False)

---

##  Reminder Methods

### Logic
- Add Reminder
- Get All Reminders
- Mark Reminder as Done

### Display
- Format One Reminder
- Format All Reminders
